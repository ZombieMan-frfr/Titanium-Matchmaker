# Titanium-Matchmaker
Titanium-Matchmaker is a universal matchmaker, and supports all of chapter 1 and chapter 2 I personally do not know about chapter 3 or higher but I will work on season support for higher seasons.
## Set up
run the install_packages.bat and make sure it installs bun and then run the start.bat and make sure you have linked to a backend and gs. The port for the matchmaker is 80 and if you have XMPP make sure both ports are different
